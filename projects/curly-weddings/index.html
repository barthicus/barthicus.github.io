Curly Weddings

Description

Personal multilingual SPA* website for Polish Weddings Planning company "Curly Weddings". CMS panel allows company owner to add new services and manage galleries. The main goal was to deliver fast, modern website that may help find more potential customers.

SPA - Single Page Apllication

Technologies

Website's layout was designed in Adobe XD and it was the base for creating responsive Bolt.CM custom theme. Bolt.CM is a developer friendly CMS that contains simple administration panel and allows users to manage data on customer's website. Bundling process was handled by Webpack:

  • compiling Scss
  • PostCSS
  • JS compilation
  • minification
